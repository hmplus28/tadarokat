# راهنمای نصب و استقرار (مدل ساده‌شده جدید)

**هیچ فایل seed وجود ندارد.** کاربران با پسوردهای ثابت در زمان نصب ساخته می‌شوند:
- admin / admin123
- mostafa / mostafa123
- fabri / fabri123
- behnaz / behnaz123
- manager / manager123
- anbar / anbar123

دیتابیس در پوشه `db/` ساخته می‌شود. اکسل ریشه پروژه در نصب اولیه ایمپورت و کش می‌شود.

## نصب (صفر تا صد)

```powershell
.\install.ps1     # یا install.bat
```

این اسکریپت:
- venv و وابستگی‌ها را نصب می‌کند
- `db/db_current.db` را با کاربران بالا می‌سازد
- اکسل ریشه پروژه را ایمپورت و کش می‌کند (برای سرعت اولین اجرا)

## اجرا

```powershell
.\run.ps1     # یا run.bat
```

## اولین کار (توسط سوپر یوزر)

1. با admin وارد شوید.
2. پنل سیستم → `primary_data_dir` را به مسیر دلخواه (محلی یا `\\server\share\...`) تنظیم کنید.
3. دیتابیس به آنجا کپی می‌شود.
4. اکسل از آنجا ایمپورت می‌شود.
5. **از این به بعد** همه چیز (دیتابیس + اکسل) فقط از همان مسیر خوانده/نوشته می‌شود.
